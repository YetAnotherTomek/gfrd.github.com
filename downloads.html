Download

You can download the latest release of the code as a zip or tar file.

Install

Binary packages for Fedora and Ubuntu will be added shortly.

Browse

You can browse the code using a web interface to the Git repository.

Clone

To check out the development version of the code, make sure you have Git installed and do:

$ git clone git://github.com/gfrd/egfrd.git

eGFRD

The current implementation of GFRD is called eGFRD (the e is for enhanced).

E-Cell

eGFRD is currently a mirror of the E-Cell project. Changes are shared between both repositories, although E-Cell is usually slightly ahead.
