Downloads

The current implementation of GFRD is called eGFRD (the e is for enhanced).

Get up and running in minutes

See the installation notes for a list of prerequisites.

E-Cell repository (Riken)

eGFRD will be included with the next release of the E-Cell project.

To check out the development version of E-Cell, make sure you have Mercurial installed and do:

$ hg clone http://t1.gsc.riken.jp/hg/epdp

eGFRD repository (Amolf)

This is currently a mirror of the E-Cell project in Git, using Hg-Git. Patches are applied to both, although the E-Cell repository is slightly more up-to-date.

To check out the development version of eGFRD, make sure you have Git installed and do:

$ git clone git://github.com/gfrd/egfrd.git

Download eGFRD releases

Instead of using Git or Mercurial, you can also download the latest release of eGFRD as a zip or tar file.

Browse

You can browse the code using the web interfaces to the eGFRD and E-Cell repositories.

eGFRD binary packages

Binary packages of eGFRD for Fedora and Ubuntu will be added shortly.
